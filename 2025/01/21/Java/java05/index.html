# 算術運算

基本的算術運算有加 (+)、減 (-)、乘 (*)、除 (/)、還有取餘數 (%)

第四行與第五行分別是整數除法與浮點數除法，當除法符號兩邊的任意一邊為小數，結果就會是以小數呈現，兩邊都是整數的話，則會忽略小數點後的數字，只保留整數的部分

同理，取餘數也分為整數餘數跟浮點數餘數

# 大小的比較

大小的比較有大於 (>)、小於 (<)、大於等於 (>=)、小於等於 (<=)

上述例子中，用到了之前提到的指定運算 (=)，所以我們先看等號右邊的內容 (5>2)，而 5>2 要表達的意思是：5 是否有大於 2，由於 5 確實有大於 2，因此 5>2 是 true (前面說過比較運算的結果是布林值)，最後再將 true 的值指派給 x 這個變數

同理，上述例子想表達的就是 5 是否有小於等於 5，並將比較後的結果指派給 y 這個變數

# 相等的比較

判斷是否相等的運算子有是否等於 (==)、不等於 (!=) 兩種

跟大小比較的概念一樣，第一行程式碼，先看等號 (第一個等號) 右邊 (5==2)，表達的意思 5 是否有等於 2，而由於 5 並不等於 2，所以 5==2 的結果是 false，再透過指定運算 (=) 將 false 的值指派給等號 (第一個等號) 左邊的 x 變數，第二行程式碼同理

要仔細分清楚 == 與 = 的差別，前者是比較運算，後者是指定運算
